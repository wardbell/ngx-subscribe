NgxSubscribe Demo

A single observable

*ngxSubscribe="interval$ as data"

{{data}}

Array of observables

*ngxSubscribe="[interval$, interval2$] as data"

{{data[0]}}, {{data[1]}}

Interval2: {{data[1]}}

Object with some observable properties

*ngxSubscribe="data as data2"

{{data2.interval}}, {{data2.name}}, {{data2.interval2}}

Note that the original name is frozen even as we toggle it: "{{data.name}}"".

No ngxSubscribe value

*ngxSubscribe

... nothing ...
